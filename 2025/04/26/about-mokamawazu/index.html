日语句型解析：～もかまわず

2025-04-26 | 日语学习
条评论

¶句型说明

“～もかまわず"可以翻译为"不管～也/即使～也/无论～也”，表示不在乎某事或某种状况而继续进行某个动作。这个表达强调即使有障碍或不理想的情况存在，也要坚持做某事。

¶接续方法

  1. 名词 + もかまわず
    例句：雪ゆきもかまわず出勤しゅっきんした。
    (不顾下雪也去上班了。)

  2. 動詞辞書形 + こと + もかまわず（注：こと可以替换为の）
    例句：失敗しっぱいすることもかまわず挑戦ちょうせんし続続つづけた。
    (不管会失败也继续挑战。)

  3. い形容詞 + こと + もかまわず（注：こと可以替换为の）
    例句：暑あついこともかまわず長袖ながそでを着きていた。
    (不管天气热，还是穿着长袖。)

  4. な形容詞 + である + こと + もかまわず（注：こと可以替换为の）
    例句：不便ふべんであることもかまわず、田舎いなかに住すんでいる。
    (不管不便，还是住在乡下。)

  5. 動詞て形 + いる + こと + もかまわず（注：こと可以替换为の）
    例句：疲つかれていることもかまわず勉強べんきょうを続つづけた。
    (尽管很疲惫，还是继续学习。)

  6. 可能形 + こと + もかまわず（注：こと可以替换为の）
    例句：遅刻ちこくできることもかまわず早はやく来きた。
    (尽管可以迟到，还是早早地来了。)

¶更多例句

  1. 雨あめもかまわず外そとで遊あそんでいる。
    (不顾下雨，在外面玩耍。)

  2. 体調たいちょうが悪わるいこともかまわず、仕事しごとに行いった。
    (尽管身体不舒服，还是去上班了。)

  3. 周まわりの目めもかまわず、大声おおごえで笑わらった。
    (不管周围人的眼光，大声笑了出来。)

  4. 危険きけんもかまわず、彼かれは川かわに飛とび込こんだ。
    (不顾危险，他跳入了河中。)

¶类似表现（与"もかまわず"相关的表达）

  1. ～てもかまわない（可以～/～也没关系）
    例句：明日あした来こなくてもかまわない。
    (明天不来也没关系。)

  2. ～なくてもかまわない（不～也没关系）
    例句：このイベントに参加さんかしなくてもかまわない。
    (不参加这个活动也没关系。)

  3. ～てもいい（可以～）
    例句：窓まどを開あけてもいいですか？
    (可以打开窗户吗？)

  4. ～なくてもいい（不～也可以）
    例句：無理むりをしなくてもいいよ。
    (你不必勉强自己。)

  5. ～ても問題もんだいない（～也没问题）
    例句：遅おそく帰かえっても問題もんだいない。
    (晚点回来也没问题。)

  6. ～なくても問題もんだいない（不～也没问题）
    例句：今回こんかいは出席しゅっせきしなくても問題もんだいない。
    (这次不出席也没问题。)

  7. ～を気きにせず（不在意～）
    例句：他人たにんの評価ひょうかを気きにせず自分じぶんの道みちを進すすんだ。
    (不在意他人的评价，走自己的路。)

  8. ～を考慮こうりょせず（不考虑～）
    例句：リスクを考慮こうりょせず投資とうしした。
    (不考虑风险就进行了投资。)

¶日常会話

A: 山田やまださん、昨日きのうの雨あめすごかったね。傘かさを忘わすれちゃったけど、どうした？

B: ああ、雨あめもかまわず走はしって帰かえったよ。服ふくがびしょ濡ぬれになっちゃった。

A: 風邪かぜを引ひいてないか心配しんぱいだね。体調たいちょうはどう？

B: 大丈夫だいじょうぶだよ。今日きょうは少すこし喉のどが痛いたいこともかまわず、仕事しごとに来きたんだ。でも、後あとでお医者いしゃさんに行いくつもりだよ。

A: 無理むりしないでね。早はやく良よくなるといいね。

¶中文

A: “山田，昨天的雨很大呢。你忘带伞了，怎么办的？”

B: “啊，我不管下雨就跑着回家了。衣服都湿透了。”

A: “担心你会感冒呢。身体状况怎么样？”

B: “没事。今天虽然嗓子有点疼，我还是来上班了。不过，待会儿打算去看医生。”

A: “别太勉强自己。希望你早日康复。”
